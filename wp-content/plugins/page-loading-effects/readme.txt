== Description ==

A simple and lightweight WP Plugin that Displays Loading Animated Graphics Until the Page is Fully Loaded. A new WP plugin that you will surely love it! They also called it Pre-Loader or Page Loader, whatever..

Go and test the [DEMO](http://tobiasahlin.com/spinkit/ "DEMO") here. Thanks to @tobiasahlin this guy. 

Best coding practices was applied to this plugin by strictly following the WP Docs, this will be available on Github soon.

= Features and Usage =

Few of the many features.
<ul>
<li>Safe to show your web content up by settings the Max Loading Duration</li>
<li>Automatically Hide the Pre-Loader when 100% resources are Loaded</li>
<li>4 Preloader Effects including default and a personalization section (for advanced users)</li>
<li>Plug and Play (unless you are playing the advanced settings)</li>
<li>Color Picker for backround and animation color</li>
<li>Animation Preview</li>
<li>Advanced Customization using HTML and CSS/CSS3</li>
<li>and more..</li>
</ul>

== Installation ==

Installation is very easy, you can either upload or install it via WP plugin section (the easy way) or Upload using FTP to wp-content/plugins/

1. Of course, Activate it via plugin section! 
1. Go to your WP Dashboard > Page Loading Effects (Just to see if it works :D)
1. The rest is just self-explanatory and You're done!
